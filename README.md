# Тема «Хребет Юрма» для WordPress

Тема для сайта туристической базы активного отдыха. Реализованы кастомные типы записей (номера, экскурсии, услуги, галерея, отзывы), управление контентом через админ-панель и кастомайзер, форма бронирования (Contact Form 7) и форма отзывов с сохранением в базу данных.

## Требования

- WordPress 6.x
- PHP 8.0 или выше
- MySQL 5.7 или выше

## Установка

1. Скопировать папку темы в `wp-content/themes/`
2. В админ-панели WordPress перейти в «Внешний вид» → «Темы»
3. Найти тему «Хребет Юрма» и нажать «Активировать»
4. Перейти в «Настройки» → «Постоянные ссылки» → выбрать вариант «Название записи» → сохранить

## Настройка после установки

### 1. Создание контента

После активации темы в админ-панели появятся разделы:
- Номера
- Экскурсии
- Услуги
- Галерея
- Отзывы

Добавьте записи в каждый раздел, заполнив все поля (цену, фото, описание и т.д.).

### 2. Настройка слотов на главной странице

Перейдите в «Внешний вид» → «Настроить»:
- **Номера на главной (4 слота)** — выберите номера для отображения
- **Экскурсии на главной (4 слота)** — выберите экскурсии
- **Услуги на главной (5 слотов)** — выберите услуги
- **Отзывы на главной (4 слота)** — выберите отзывы

### 3. Настройка формы бронирования

Установите и активируйте плагин Contact Form 7. Создайте форму с полями: имя, телефон, дата заезда, дата выезда, сообщение. Полученный шорткод вставьте в модальное окно (файл `footer.php`).
